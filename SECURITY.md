# Security

PandaOS is **non-custodial**. The agent has no signing path: it reads on-chain data and drafts
actions, and every transaction is approved by you in your own wallet. PandaOS never holds private
keys, and any LLM key or memory file lives only in your browser's local storage.

## Reporting

Found a vulnerability? Please report it privately rather than opening a public issue. Open a
[security advisory](https://github.com/mycopok/panda-os/security/advisories/new) or email the
maintainer. We will respond as fast as we can.

Treat all agent output as research. Always review any drafted transaction before you sign it.
